# 🌌 Black Hole Simulator (Educational Project)

> An educational C++ and OpenGL project that builds a black hole simulator from first principles.

This repository is **not just a black hole visualization**.

The goal of this project is to explain **how black holes can be simulated in real-time**, while teaching the mathematics, physics, and computer graphics concepts behind every implementation.

Every milestone introduces a new feature together with the theory used to build it.

---

# 🎯 Project Goals

This project aims to teach:

- Modern C++
- OpenGL Rendering Pipeline
- GLSL Shader Programming
- Linear Algebra for Computer Graphics
- 3D Camera Mathematics
- Mesh Generation
- Numerical Methods
- Schwarzschild Geometry
- General Relativity (Simplified)
- Real-Time Graphics Programming

Instead of jumping directly into advanced visual effects, every feature is implemented step by step with explanations.

# 🚀 Educational Philosophy

Many graphics projects focus only on producing beautiful visuals.

This project takes a different approach.

Every feature is implemented from scratch while explaining:

- Why the mathematics works
- Why the physics behaves that way
- Why the rendering technique is chosen
- How everything connects inside an OpenGL application

The objective is to make the repository useful not only as software, but also as a learning resource for students interested in Computer Graphics, Mathematics, Physics, and C++.
